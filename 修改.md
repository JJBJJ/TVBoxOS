# TVBoxOS 弹幕搜索扩展修改说明

项目代码来源俊佬：[TVBoxOS](https://github.com/q215613905/TVBoxOS)

本次修改主要新增了一个 **弹幕搜索自定义扩展接口**。App 端负责提供弹幕搜索按钮、收集当前影片名称和集数信息，并异步传递给外部 dex；具体弹窗、搜索、推送等业务逻辑由 dex 开发者自行实现。

### 2. `VodController.java`

### 修改内容

对播放器控制器中的弹幕搜索按钮 `mDanmuSearchUiBtn` 进行声明、绑定，并增加点击和长按事件。

### 新增控件声明

```java
TextView mDanmuSearchUiBtn;
```

### 新增控件绑定

```java
mDanmuSearchUiBtn = findViewById(R.id.danmu_search_ui);
```

### 新增逻辑

- 普通点击：调用弹幕搜索普通点击接口。
- 长按：调用弹幕搜索长按接口。
- 点击和长按后都调用 `hideBottom()`，隐藏底部控制栏，保持播放器 UI 行为一致。

核心逻辑：

```java
mDanmuSearchUiBtn.setOnClickListener(v -> {
	if (listener != null) {
		listener.searchDanmuUi(false);
		hideBottom();
	}
});

mDanmuSearchUiBtn.setOnLongClickListener(v -> {
	if (listener != null) {
		listener.searchDanmuUi(true);
		hideBottom();
	}
	return true;
});
```

同时在 `VodControlListener` 中新增接口方法：

```java
void searchDanmuUi(boolean longClick);
```

### 作用

播放器 UI 只负责按钮事件，不直接实现弹幕搜索业务。点击类型通过 `longClick` 区分：

- `false`：普通点击
- `true`：长按

## 二、播放页调用修改

### 3. `PlayActivity.java`

文件位置：

[`app/src/main/java/com/github/tvbox/osc/ui/activity/PlayActivity.java`](app/src/main/java/com/github/tvbox/osc/ui/activity/PlayActivity.java)

### 修改内容

实现 `VodControlListener` 中新增的 `searchDanmuUi(boolean longClick)` 方法。

### 新增逻辑

从当前播放信息中取出：

- `mVodInfo.name`：当前影片名称
- `series.name`：当前集数名称

然后调用 `ApiConfig` 异步分发。

核心逻辑：

```java
public void searchDanmuUi(boolean longClick) {
	VodInfo.VodSeries series = mVodInfo == null ? null : getCurrentSeries(mVodInfo.playFlag, mVodInfo.playIndex);
	ApiConfig.get().searchDanmuUi(mVodInfo == null ? "" : mVodInfo.name, series == null ? "" : series.name, longClick);
}
```

### 作用

在 Activity 播放模式下，把当前影片名、当前集数和点击类型传递给弹幕扩展接口。

## 三、异步调用修改

### 5. `ApiConfig.java`

文件位置：

[`app/src/main/java/com/github/tvbox/osc/api/ApiConfig.java`](app/src/main/java/com/github/tvbox/osc/api/ApiConfig.java)

### 修改内容

新增弹幕搜索专用异步线程池和统一入口方法。

### 新增代码

新增单线程执行器：

```java
private final ExecutorService danmuSearchExecutor = Executors.newSingleThreadExecutor();
```

新增异步调用方法：

```java
public void searchDanmuUi(String name, String episode, boolean longClick) {
	danmuSearchExecutor.execute(() -> {
		try {
			jarLoader.searchDanmuUi(name, episode, longClick);
		} catch (Throwable th) {
			LOG.e("ApiConfig searchDanmuUi error: " + th.getMessage());
		}
	});
}
```

### 作用

弹幕搜索扩展调用不会阻塞播放器主线程。即使 dex 侧执行耗时逻辑，也不会卡住播放 UI。

## 四、外部 dex 方法加载修改

### 6. `JarLoader.java`

文件位置：

[`app/src/main/java/com/github/catvod/crawler/JarLoader.java`](app/src/main/java/com/github/catvod/crawler/JarLoader.java)

### 修改内容

增加对外部 dex 中 `com.github.catvod.spider.Danmaku` 类的反射加载和方法缓存。

### 新增字段

```java
private final ConcurrentHashMap<String, Method> danmuClickMethods = new ConcurrentHashMap<>();
private final ConcurrentHashMap<String, Method> danmuLongClickMethods = new ConcurrentHashMap<>();
```

### 新增加载逻辑

在 dex/classloader 加载完成后，尝试反射查找：

```java
Class<?> danmaku = classLoader.loadClass("com.github.catvod.spider.Danmaku");
Method clickMethod = danmaku.getMethod("onClick", String.class, String.class);
Method longClickMethod = danmaku.getMethod("onLongClick", String.class, String.class);
```

并分别缓存到：

```java
danmuClickMethods.put(key, clickMethod);
danmuLongClickMethods.put(key, longClickMethod);
```

### 新增调用方法

```java
public void searchDanmuUi(String name, String episode, boolean longClick) {
	try {
		ConcurrentHashMap<String, Method> methods = longClick ? danmuLongClickMethods : danmuClickMethods;
		Method method = methods.get(recentJarKey);
		if (method == null) method = methods.get("main");
		if (method == null) return;
		method.invoke(null, name, episode);
	} catch (Throwable th) {
		Log.i("JarLoader", "echo-searchDanmuUi error key=" + name + ", episode=" + episode + ", longClick=" + longClick + ", msg=" + th.getMessage());
	}
}
```

### 作用

App 不直接依赖 dex 侧具体实现，只约定类名和方法名。开发者只要在 dex 中实现对应静态方法即可。


## 六、dex 开发者对接方式

dex 侧只需要实现固定类：

```java
package com.github.catvod.spider;

public class Danmaku {

	public static void onClick(String name, String episode) {
		// 普通点击弹幕搜索按钮时调用
	}

	public static void onLongClick(String name, String episode) {
		// 长按弹幕搜索按钮时调用
	}
}
```

参数说明：

- `name`：当前影片名称
- `episode`：当前集数名称
